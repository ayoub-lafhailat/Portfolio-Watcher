# PortfolioWatcher

## Overview

PortfolioWatcher is a full-stack ASP.NET application designed to help users manage and monitor their stock portfolios and trades in a structured and user-friendly way.

The project was developed with a strong focus on software engineering principles such as layered architecture, separation of concerns, SOLID principles, dependency injection, validation, and maintainability.

Users can create portfolios, register trades, manage stock symbols, and track portfolio performance through an organized web interface.

---

# Features

- Create and manage investment portfolios
- Register buy and sell trades
- Track portfolio holdings
- Profit and loss calculations
- Stock symbol management
- Input validation and exception handling
- Layered application architecture
- Database persistence with SQL Server
- Responsive Bootstrap-based interface

# Architecture

The application was built using a layered architecture to improve maintainability, scalability, and separation of responsibilities.

The architecture consists of:

- Presentation Layer
- Service Layer
- Repository Layer
- Domain Layer
- Database Layer

The project also uses dependency injection to decouple components and improve testability.
